# 🚀 Nexbit - Video Game Store

## 📌 Descripción
**Nexbit** es una plataforma web desarrollada como parte del **Trabajo Práctico N°1** de la materia **Programación 3**. El proyecto ha sido construido utilizando **HTML5 y CSS3**, con un enfoque en el diseño "Dark Mode" y una estética futurista inspirada en el mundo del gaming. 

El objetivo principal fue aplicar conceptos fundamentales de desarrollo frontend, estructuración de layouts mediante Flexbox y personalización de estilos avanzados.

## 🛠️ Tecnologías utilizadas
* **HTML5:** Estructura semántica del sitio.
* **CSS3:** Estilos personalizados, variables, Flexbox y animaciones de transición.
* **SVG:** Iconografía escalable para la sección de contacto y equipo.

## 📄 Páginas del sitio
El sitio está compuesto por las siguientes secciones:
* 🏠 **Home** (`index.html`): Presentación principal y novedades.
* 🛠️ **Servicios**: Catálogo de soluciones y preventa.
* 👥 **Equipo**: Perfiles de los fundadores y especialistas.
* 📝 **Pedido**: Formulario interactivo para adquisiciones.
* 📞 **Contacto**: Información de ubicación y soporte técnico.
* ❓ **FAQ**: Preguntas frecuentes sobre envíos y garantías.
